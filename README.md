Getting Started:

1. Install grunt requirements:
  npm install -g grunt
  npm install grunt grunt-contrib-concat grunt-contrib-qunit \
    grunt-contrib-jshint grunt-blanket-qunit --save-dev

2. Build (will run tests / check coverage is over 90% / jshint):
  Just run grunt;

  grunt
